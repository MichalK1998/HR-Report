The file 'HR Report' contains the created report. The subject of the report is the employee situation in the company. It contains basic information about employees such as name, surname, age, but also information related to the position in the company such as monthly earnings. In the file you will find 3 tables with data. The 'HR analytics data' table contains detailed information about employees. The 'HR employee data' table contains names, surnames and id numbers assigned to employees. In the 'Measures' table, you will find the calculated measures used in creating the report.
